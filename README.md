# 👨‍💼 Employee Management System

A console-based **Employee Management System** built in **C++** as a 3rd semester Data Structures & Algorithms project at Air University. Uses file handling for persistent data storage and implements core DSA concepts for efficient record management.

---

## ✨ Features

- ➕ **Add Employees** — Register new employee records
- 🔍 **Search Employees** — Find records by ID or name
- ✏️ **Update Records** — Modify existing employee information
- ❌ **Delete Employees** — Remove records from the system
- 🔐 **Admin Authentication** — OTP-based admin login system
- 💾 **File Handling** — Data persists across sessions using text files

---

## 🧠 DSA Concepts Used

- **File-based storage** — Reading and writing structured data
- **Linear Search** — Finding employee records efficiently
- **CRUD Operations** — Core data manipulation logic
- **Modular Programming** — Clean function-based structure

---

## 🛠️ Tech Stack

- **Language:** C++
- **Concepts:** Data Structures, File Handling, OTP Authentication
- **IDE:** Visual Studio / Code::Blocks

## ⚙️ How to Run

1. Clone the repository:
```bash
   git clone https://github.com/muhammadbilalkhalidmbk/employee-management-system-cpp.git
```
2. Compile the source file:
```bash
   g++ "Employee Management System.cpp" -o ems
```
3. Run the program:
```bash
   ./ems
```
